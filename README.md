# 寿险意外险并税管理系统

欢迎使用寿险意外险并税管理系统！

## 项目简介

本项目是一个寿险意外险并税管理系统，用于管理员工寿险和意外险的并税相关信息，包括保险信息管理、并税详情记录、状态追踪等功能。

## 项目结构

```
.
├── web/           # 前端项目目录
├── api/           # 后端项目目录
│   ├── insurance_tax_management.sql    # 寿险意外险并税管理列表表
│   └── tax_detail.sql                  # 并税详情表
└── README.md      # 项目说明文档
```

## 功能模块

### 1. 寿险意外险并税管理列表
- 员工保险信息管理
- 保险年度管理
- 保险期限管理
- 并税状态追踪
- 推送状态管理
- 员工入职/离职信息关联

### 2. 并税详情管理
- 并税单号管理
- 并税日期范围记录
- 保险天数和并税天数统计
- 并税状态和推送状态追踪

## 数据库表设计

### insurance_tax_management（寿险意外险并税管理列表表）
- 记录员工的基本保险信息
- 包含并税状态和推送状态
- 关联员工入职、离职、组织等信息

### tax_detail（并税详情表）
- 记录并税单的详细信息
- 统计保险总天数和并税天数
- 追踪并税和推送状态

## 快速开始

### 环境要求
- MySQL 5.7+
- Node.js（前端）
- Java/Spring Boot 或其他后端框架（后端）

### 数据库初始化
```bash
# 执行数据库脚本
mysql -u root -p your_database < api/insurance_tax_management.sql
mysql -u root -p your_database < api/tax_detail.sql
```

## 开发说明

- `web/` 目录用于存放前端代码
- `api/` 目录用于存放后端代码和数据库脚本
- 数据库脚本已准备好，可直接在 MySQL 中执行
